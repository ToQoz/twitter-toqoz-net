### 頭骨API
頭骨名義でつぶやけます
http://twitter.toqoz.net

### 設置方法
    $ git git://github.com/ToQoz/twitter-toqoz-net.git
    $ mkdir logs
    $ mkdir tmp
    $ cp config/config.yml.sample config/config.yml
    # 自分のaccess_tokenとか入れてください
    $ vi config/config.yml
    $ bundle install
    $ ruby app.rb
